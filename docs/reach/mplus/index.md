## Package contents

In the box:

* 1 x Reach M+ module

* 1 x USB cable 

* 1 x USB-OTG cable

* 1 x JST-GH 6-pin to jumper pin cable 

* 1 x JST-GH 5-pin to jumper pin cable

Optionally: 

* 1 x Tallysman multi-GNSS antenna
